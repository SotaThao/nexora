# Proposal: Verify Staff Registration Portal

## Why
We need to verify that the staff registration portal and personal dashboard work correctly under all simulated and data-driven conditions, including linked businesses, payout methods, QR code generation, and notifications.

## What Changes
We will run both unit tests and E2E tests for the staff portal and dashboard.

## Capabilities
- `verify-staff-registration-portal`: Verifies step-by-step registration wizard, live profile searching, OTP codes verification, payout toggles, and personal dashboard views.

## Impact
No direct application code modifications are required for this verification change.
