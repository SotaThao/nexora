
# NEXORA TOUCH by VLINKPAY


## Implementation & Development Specification

NEXORA TOUCH is a QR-first + NFC optional Smart Tip & Review Platform for nail salons, restaurants, cafes, spas, bars, hotels, and service businesses.


## 1. Core Product Positioning

QR Code is the default mass-market solution.

NFC is an optional premium upgrade.

Main use cases: Tip, Review, Staff Profile, Merchant Analytics.

Integrated with NEXORA digital identity ecosystem.


## 2. Customer Flow

Scan QR or Tap NFC

Select Staff

Choose Tip Amount

Choose Payment Method

Payment Success

Leave Review

Redirect to Google/Yelp for positive reviews
